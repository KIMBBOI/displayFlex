## 주요 기능
### 공통
 - 로그인
 - 영화 목록 조회
 - 영화 조건 검색(장르, 관람등급을 조합해서 조건에 맞는 영화 검색)
    - 장르별
    - 관람등급별
### 사용자
 - 예매하기
 - 리뷰 작성/조회
